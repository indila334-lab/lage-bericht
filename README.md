## Current project summary

We are building a physical home Severin through a XiaoZhi cube:

- XiaoZhi cube is the body: screen, mic, speaker, buttons, Wi-Fi/4G, faces.
- Honor Magic 6 Pro with Termux is the practical local node and bridge.
- GitHub is durable memory and code storage.
- xiaozhi.me is the current cube console.
- Mordashka stores face and emotion assets.
- Pluton/OpenCloud/Copilot can help with code but are not permanent runtime brains.
